The icsv class¶

class icsv.icsv(headers, delimiter=', ')¶

The icsv class encapsulates the data contained in a single CSV file.

A CSV consists of zero or more rows of information where each row has one or more named columns. The names of the columns are referred to as headers.

The icsv class provides an interface for easily adding new data to a CSV, writing the data to a CSV file, and creating an icsv from a file.

The Writer class¶

class icsv.Writer(filename, headers, delimiter=', ', useHeaders=True, overwrite=True)¶

The Writer class provides an interface for writing a CSV file.

This class can be used to write a pre-existing icsv.icsv object to a file, or be used to write data to a filename in real time.

The Writer class provides a wrapper to the icsv.icsv class which will immediately write new rows to the CSV file.

__init__(filename, headers, delimiter=', ', useHeaders=True, overwrite=True)¶
Parameters:
  • filename (string) – The filename for the CSV file to write
  • headers (list of string) – The list of column headers for this CSV file
  • delimiter (string) – The CSV delimiter to use
  • useHeaders (bool) – True will write the headers to the first line of the created CSV file, False will not
  • overwrite (bool) – True will overwrite existing files, False will not
Raises Exception:
 

If overwrite is False, and the file already exists
